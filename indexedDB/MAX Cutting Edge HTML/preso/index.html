LocalStorage

  • Very simple API: getItem, setItem, removeItem, clear
  • Simple values only, but you can encode
  • Session version
  • Up to 2.49 megs (not 2.5, oh no, 2.49!)

Example


//API way
localStorage.setItem("name", "value");
var x = localStorage.getItem("name");
localStorage.removeItem("name");
localStorage.clear();

//Non API way
localStorage["name"] = "value";
var x = localStorage["name"];
delete localStorage["name"];
for(var x in localStorage) delete localStorage[x];

IndexedDB

IndexedDB is an API for client-side storage of significant amounts of structured data and for high performance searches on this data using indexes.

Credit: Mozilla Developer Network

Size Limits

  • Firefox - no limit, but prompted for 'large' data
  • Chrome - User is prompted once
  • Take away - you can store a butt load, but the user will know

Know SQL?

High Level Overview

  • Asynchronous API
  • Object stores contain data (think tables)
  • Fetch API based on keys (think PKs), with full create, read, update, delete support
  • Fetch by property (index), and by range
